Для запуска срипта необходимо добавить класс js-selector к тэгу select для которого Вы хотите его применить

Необходимые data-атрибуты для тэга select - data-name (не должны повторяться, тк от них формируются id), data-selectortext (для текста, который будет в имени селекта. Есть значение по умолчанию)
Для вложенных option нужны: value и data-level

ВАЖНО: без value у option не получится выстроить связь для передачи информации в select и связать option нижнего уровня с верхним
       без data-level не получится выстраить их в ряд и связать

Примечания:
1. Чтобы убрать автоматический выбор по нажатию на верхний блок, нужно закоментить строку 236 (над ней пометка //++++++++++Важная строчка 1+++++++++++++)
2. функция selectorInfoCreator принимает в себя 2 аргумента (массив выбранныз options, максимальная длинна в символах к которой будут добавлены 3 точки) по умолчанию второй параметр отключени (false) и строка не будет обрезаться
3. Функция selectorBlocksCreator принимает в себя 3 аргумента (имя которое будет указнно вверху, сам селектор ссылкой на тэг, name от select на основании которого будут формироваться id) первый необязательный
4. Функция selectSelectedFromLevels принимает в себя 3 аргумента (массив option из select, индес в массиве option от которого будет проверка (не включая его), уровень выше которого не будут браться значения)
5. addClass(), createChild(), changeCounter(), changeText(), changeTitle() - простые и не требуют объяснения
6. fireEvent() - используется для создания кастомного ивента, чтобы при изменении на select срабатывал "change"
7. fakeSelectClick() - принимает в себя ивент, который происходит при нажатии на фейковый option. Если не находит нужный ему параметр (id родительского элемента), то пробует подняться на 1 уровень выше от event.target
8. changeSelect() - принимает в себя фейвый селект (div с id формата list-for-|name из select| и изменяет его, а так же оригинальный select
9. showOptionsList() - открывает список фейковых option при ивенте 'click'
10. getSelectedOptionsArr(ссылка на select, ?ключи для option которые сложит в массив, ?вернет информацию о массиве option) - принемает select и возвращает массив в зависимости от параметров
